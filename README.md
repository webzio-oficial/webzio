```
├── index.html        ← Tu página web completa

```

## Cómo subir a Netlify (gratis, 2 minutos)
1. Ve a https://netlify.com y crea cuenta gratis
2. En el dashboard haz clic en "Add new site" → "Deploy manually"
3. Arrastra la carpeta `webzio-project` completa al área de drop
4. Netlify te da una URL como `webzio.netlify.app` al instante
5. En Site Settings → Domain → Add custom domain → escribe `webzio.co`

## Cómo usar en VS Code
1. Abre VS Code
2. File → Open Folder → selecciona `webzio-project`
3. Instala la extensión "Live Server" (botón Extensions, busca Live Server)
4. Clic derecho en index.html → "Open with Live Server"
5. Se abre en tu navegador con recarga automática al guardar
